# WebSocket Chat App

A simple real-time chat app using Node.js and native WebSocket.

## How to Run

1. Run `npm install`
2. Run `npm start`
3. Open `http://localhost:3000` in your browser

Test by opening in two tabs or devices.

## Tech Stack

- Node.js (HTTP server)
- WebSocket (`ws` package)
- HTML/CSS/JavaScript frontend
